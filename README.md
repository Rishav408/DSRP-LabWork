# DSRP - Data Science using R Programming Lab Work

A comprehensive collection of lab assignments and datasets for the **Data Science using R Programming (DSRP)** course. This repository includes Jupyter Notebooks and structured VIVA preparation guides covering R fundamentals, statistical analysis, and machine learning.

## 🚀 Lab Assignments Overview

| Lab ID | Topic | Notebook Link | Core Concepts |
| :--- | :--- | :--- | :--- |
| **Lab 1** | R Programming Fundamentals | [Open Notebook](./R-LAB1-Basics%20R.ipynb) | Data Structures (Vectors, Lists, Frames), Functions, Loops |
| **Lab 2** | Statistical Hypothesis Testing | [Open Notebook](./R-LAB2-Hypothesis%20Testing.ipynb) | T-Tests, ANOVA, Chi-Square, Correlation |
| **Lab 3** | Association Rule Mining | [Open Notebook](./R-LAB3-Association-Rule-using-Apriori.ipynb) | Apriori Algorithm, Support, Confidence, Lift |
| **Lab 4** | Linear Regression | [Open Notebook](./R-LAB4-Linear%20Regression.ipynb) | Simple & Multiple Linear Regression, Residual Analysis |
| **Lab 5** | Logistic Regression | [Open Notebook](./R-LAB5-Logistic%20Regression.ipynb) | Binary Classification, Evaluation (ROC, AUC, Confusion Matrix) |
| **Lab 6** | Naive Bayesian Classifier | [Open Notebook](./R-LAB6-Naive-Bayesian-Classifier.ipynb) | Conditional Probability, Bayesian Inference |
| **Lab 7a** | Decision Trees | [Open Notebook](./R-LAB7a-Recursive-Partitioning.ipynb) | Recursive Partitioning, Tree Visualization, Gini/Entropy |
| **Lab 7b** | K-Means Clustering | [Open Notebook](./R-LAB7b-K-Means-Clustering.ipynb) | Unsupervised Learning, Elbow Method, Centroids |
| **Lab 8** | Random Forest | [Open Notebook](./R-LAB8-Random-Forest.ipynb) | Ensemble Learning, Bagging, Variable Importance |

## 📊 Datasets Used
- **mtcars**: Motor Trend Car Road Tests.
- **Boston**: Housing values in suburbs of Boston.
- **GermanCredit**: Credit risk data for classification.
- **Groceries**: Transactional dataset for Association Rules.

## 🛠 Prerequisites
- **R (>= 4.0)**
- **Required Packages**: `MASS`, `caret`, `pROC`, `ggplot2`, `arules`, `arulesViz`, `e1071`, `rpart`, `rpart.plot`, `randomForest`, `factoextra`, `cluster`.
